Brooklyn College Foundation Setup

This is a small page with a number of rows, explaining how to setup the most basic site layout, as well as provide links to the various JS code elements, etc.

All other styling happens of a per project basis. The use of the PANEL style is for visualization on this page only.

The Grid

The default BC grid has 12 columns, and max-width of 1080px. Each row (this is a row) spans the full width of the grid, and contains content elements that use column width styles for size and arrangement at each breakpoint. This is a 12-column content element at the large size. The grid has zones of presentation:

  • Xlarge - above 1441px
  • Large - between 1440 and 849px
  • Medium - between 848px and 641px
  • Small - smaller than 640px

Column Content Elements

Rows are filled with content elements that can respond to the presentation zones. Add styles to these elements:

  • columns - add to all content elements
  • large-X
  • medium-X
  • small-X

If there is no definition for the specific zone, the content element will fill the grid 100%. This allows for some elements to maintain a column definition while other abandon it as screens change. You can even nest rows within content elements, like this:

Media Queries

Specific content elements can also break independently of the grid with media queries:

  • Desktop - 850px or more
  • iPadPortrait - less than 850px
  • Phone - less than 768px
  • Five00 - less than 500px
